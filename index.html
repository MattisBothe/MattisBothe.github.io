Was ist Rusty?

Rusty ist ein KI‑gestützter Discord‑Bot mit Persönlichkeit. Er wurde als privates Heimprojekt entwickelt und wächst ständig weiter.

Features

  • Antwortet intelligent auf Nachrichten
  • Hat eine eigene Persönlichkeit
  • Kann erweitert und angepasst werden
  • Open‑Source auf GitHub
